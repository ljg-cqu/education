## 一、前言与使用说明

### 1.1 本指南的定位

本指南是一份**面向朴素家长的教育全景手册**，旨在帮助您：

1. **透彻理解**四川省及成都市的教育体系和政策框架
2. **全面掌握**从幼儿园到大学各阶段的关键决策点
3. **了解跨省选项**，特别是回贵州参加高考的路径
4. **知晓少数民族教育政策**，特别是布依族相关加分和专项计划
5. **获得远期规划视角**，不仅关注当前升学，更关注孩子长远发展

### 1.2 与已有指南的关系

本指南泛化自《成都成华区择校与升学规划指南》（v2.1），但**不局限于成华区**。成华区内容作为典型案例保留，同时扩展至成都各区、四川省及跨省（贵州）层面。成华区指南的§6.6.3四种路径全面对比、§6.6.3a中考分流与升学结果分析、§6.7关键约束清单为本指南§十四路径评估矩阵和§十五战略决策框架的区级细化版。

## 二、四川省教育体系全景概览

### 2.1 学制总览

四川省实行**六三学制**（小学6年+初中3年+高中3年），与全国大多数省份一致。

| 阶段 | 年限 | 年龄范围 | 性质 | 关键节点 |
|:---|:---:|:---:|:---|:---|
| 幼儿园 | 3年 | 3-6岁 | 非义务教育 | 小班/中班/大班 |
| 小学 | 6年 | 6-12岁 | 义务教育 | 划片入学、免试就近 |
| 初中 | 3年 | 12-15岁 | 义务教育 | 小升初摇号/划片、中考 |
| 高中 | 3年 | 15-18岁 | 非义务教育 | 中考录取、新高考选科 |
| 大学 | 4年/5年 | 18+ | 高等教育 | 高考志愿填报 |

### 2.2 教育资源分布（省/市/区三级）

#### 省级层面

四川省共有21个市（州），教育资源集中在**成都、绵阳、德阳**三个核心城市。其中：

- **成都**：省会，教育资源最丰富，拥有"四七九"（石室中学、成都七中、树德中学）等顶级名校
- **绵阳**：四川教育第二城，绵阳中学、南山中学等实力强劲
- **民族地区**：甘孜州、阿坝州、凉山州（"三州"）享有民族地区加分政策

#### 市级层面（成都市）

成都市共划分**16个招生区域**，其中"5+2"区域（天府新区、高新区、锦江区、青羊区、金牛区、武侯区、成华区）组成一个统一招生区域，其余各区（市）县各自独立。[[来源]](https://www.zzto.com.cn/news-id-4949.html)

| 区域 | 教育实力定位 | 特色 |
|:---|:---|:---|
| **青羊区** | 第一梯队 | 名校密集，石室联中、树德实验等"三小联中" |
| **锦江区** | 第一梯队 | 七中育才、盐道街中学等优质资源 |
| **金牛区** | 第二梯队 | 指标到校名额最多（2024年1510个）[[来源]](https://www.zizzs.com/gk/zhongkao/197398.html) |
| **成华区** | 第二梯队 | 指标到校第二（2024年1461个），列五/华西/49中等 |
| **武侯区** | 第二梯队 | 川大附中、西北中学等 |
| **天府新区** | 新兴梯队 | 指标名额较少（747个），但民办学校校内直升比例高 |
| **高新区** | 新兴梯队 | 七中高新校区、市属名校分校聚集 |

#### 区级层面（成华区典型案例）

成华区作为典型案例，拥有公办小学30+所、民办小学5所、公办初中22所、民办初中6所、公办高中9所、民办高中1所。详细名单和梯队划分见《成都成华区择校与升学规划指南》§1.1-§1.3；万汇小学出发的四种升学路径全面对比见该指南§6.6.3，中考分流与升学结果分析见§6.6.3a，关键约束清单见§6.7。

### 2.3 关键政策术语解释

| 术语 | 含义 |
|:---|:---|
| **5+2区域** | 天府新区、高新区、锦江区、青羊区、金牛区、武侯区、成华区，统一招生区域 |
| **大摇号** | 市直属学校（四七九+二中）电脑随机录取，面向全市小学毕业生 |
| **小摇号** | 区属公办学校电脑随机录取 |
| **划片入学** | 按户籍/居住地就近分配公办学位 |
| **指标到校** | 优质高中将部分招生名额直接分配到初中学校，无需中考即可入学 |
| **统招生** | 户籍和学籍均在同一招生区域的考生 |
| **调招生** | 学籍在招生区域但户籍不在的考生 |
| **随迁子女** | 非本地户籍、跟随父母在流入地就读的子女 |
| **3+1+2** | 新高考模式：3门统考+1门首选（物理/历史）+2门再选 |
| **赋分制** | 再选科目按等级转换赋分，非原始分 |
| **院校专业组** | 新高考志愿填报单位，一个院校可设多个专业组 |
| **三统一** | 少数民族加分条件：户籍、学籍、实际就读地三年一致 |
| **项目班** | 高中特色班级（科技创新/强基/人文/航空/球类预备队等），第一批次录取 |
| **职普融通** | 中职与普高融通班，高一后可互转，2025年成都新增录取批次 |
| **强基计划** | 面向基础学科的拔尖人才培养计划，39所双一流高校，高考85%+校考15% |
| **综合评价** | 高校综合高考成绩、学考成绩、校测成绩、综合素质评价的多元录取模式 |
| **保送生** | 无需参加高考，经推荐和考核直接录取（竞赛/外语/运动员/公安英烈子女） |
| **少年班** | 面向低龄拔尖学生的特殊招生（中科大/西交大/东南大学），跳级升大学 |
| **"11+2"区域** | 成都市统一招生区域，即11个行政区（锦江区、青羊区、金牛区、武侯区、成华区、龙泉驿区、青白江区、新都区、温江区、双流区、郫都区）+ 2个新区（天府新区、高新区）。比"5+2"范围更大，大摇号面向"11+2"区域户籍 |

### 2.4 全路径升学矩阵总览

> **这是本指南最核心的章节之一。** 很多家长只知道"中考→高考"一条路，实际上从幼儿园到大学，每一步都有**多条并行路径**。下面用一张矩阵图展示所有可能的升学路径，帮助您建立全局视野。

#### 幼儿园→小学

| 路径 | 方式 | 确定性 | 关键约束 |
|:---|:---|:---:|:---|
| 公办划片 | 按户籍/居住地就近分配 | ★★★★★ | 户籍在学区内 |
| 民办摇号 | 统一报名、电脑随机录取 | ★★ | 报名超计划才摇号 |
| 民办一贯制直升 | 已在本校幼儿园→直升小学 | ★★★★ | 需在民办一贯制学校就读 |
| 随迁子女统筹 | 教育局统筹安排 | ★★★ | 需提前申请（4月） |

#### 小学→初中（小升初）

| 路径 | 方式 | 确定性 | 关键约束 | 适合谁 |
|:---|:---|:---:|:---|:---|
| 单校划片（对口直升） | 小学直接对口初中 | ★★★★★ | 学籍在对口小学 | 所有学生，最省心 |
| 多校划片（小摇号） | 片区内多所初中摇号 | ★★★ | 学籍在片区内 | 愿意接受随机分配 |
| 大摇号（市直属） | 四七九+二中电脑随机 | ★☆ | "11+2"区域户籍 | 愿意博弈名校（中签率1-5%） |
| 民办初中摇号 | 统一报名、电脑随机 | ★★ | 报名超计划才摇号 | 追求民办教育资源 |
| 民办一贯制直升 | 小学→初中直升 | ★★★★ | 在民办一贯制学校就读 | 已在民办就读且满意 |
| 艺体特长生（历史渠道） | 专业测试+文化成绩 | ★★★ | 需有艺术/体育特长 | 有特长的学生（注意：此渠道正在收紧） |
| 七中育才学道分校（特殊） | 五年级后报名、摇号 | ★★ | 锦江区户籍/学籍 | 跳过六年级、提前衔接初中 |

#### 初中→高中（中考）

| 路径 | 方式 | 确定性 | 关键约束 | 适合谁 |
|:---|:---|:---:|:---|:---|
| 指标到校（市级统分） | 四七九等名校分配名额 | ★★★★ | 校内排名+中考达普高线 | 校内排名前列的学生 |
| 指标到校（区级指标） | 区属优质高中分配名额 | ★★★★★ | 校内排名（部分免中考） | 校内排名前列、目标区内高中 |
| 指标到校（校内直升） | 完全中学初中→高中直升 | ★★★★★ | 在本校初中部就读 | 已在完全中学就读且满意 |
| 中考统招（第一批次） | 艺体特长生/项目班/航空班 | ★★★ | 需特长或特殊资格 | 有特长/特殊志向的学生 |
| 中考统招（第二批次） | 公办普高平行志愿录取 | ★★★ | 中考分数+户籍在5+2 | 主流通路，凭分数择校 |
| 中考调招 | 面向非5+2户籍考生 | ★★ | 名额少于统招 | 户籍不在5+2的学生 |
| 民办高中 | 自主招生/中考成绩 | ★★★ | 学费高昂 | 成绩好+经济条件好 |
| 职普融通班 | 中职与普高融通 | ★★★ | 中考达职普融通线 | 分数在普高线边缘 |
| 中职学校 | 免试/按分数录取 | ★★★★ | 分数较低 | 分数较低或有意走职业技能路线 |

#### 高中→大学（高考）

| 路径 | 方式 | 难度 | 关键约束 | 适合谁 |
|:---|:---|:---:|:---|:---|

## 六、高中阶段

### 6.1 中考政策

#### 统招生 vs 调招生

| 类别 | 条件 | 录取优先级 |
|:---|:---|:---|
| **统招生** | 户籍和学籍均在"5+2"区域 | 高（可报所有面向5+2招生的学校） |
| **调招生** | 学籍在"5+2"但户籍不在 | 中（部分学校不招收调招生，或名额较少） |

> **来源**: [中考政策解读](https://www.zzto.com.cn/news-id-4949.html)

#### 跨区中考

- 中考报名以**户籍为准**
- 若户籍在成华区、学籍在青羊区，可在青羊区参加中考，也可回成华区报名
- 回户籍地报名需提前办理相关手续

### 6.3 跨省高中选择路径

| 路径 | 说明 | 优势 | 劣势 |
|:---|:---|:---|:---|
| **成都读高中→四川高考** | 最常规路径 | 四川高考录取线相对合理、熟悉政策 | 四川高考竞争激烈 |
| **回贵州读高中→贵州高考** | 户籍在贵州则可回原籍高考 | 少数民族加分（布依族）、贵州录取线可能更低 | 需适应新环境、教材可能不同 |
| **成都读高中→回贵州高考** | 在成都读高中但回贵州参加高考 | 享受成都优质教育资源+贵州加分 | 需确认贵州高考报名条件（三统一等） |

> **注意**：贵州少数民族加分要求"三统一"——高中阶段三年户籍、学籍和实际就读地在同一区域。如在成都读高中再回贵州高考，可能**不满足三统一条件**，无法享受加分。详见§八和§九。

### 7.1 四川新高考（3+1+2模式）

### 7.2 志愿填报与录取

#### 志愿填报单位：院校专业组

新高考采用**"院校+专业组"**模式，以平行志愿方式投档。

- 一个院校可设多个专业组（如"四川大学物理组01"和"四川大学历史组02"）
- 每个专业组内包含若干专业
- 考生按专业组填报志愿，投档到专业组后再进行专业分配

#### 录取规则

| 规则 | 说明 |
|:---|:---|
| **分列计划** | 按物理类、历史类分列招生计划 |
| **分开划线** | 物理类和历史类分别划定分数线 |
| **分别录取** | 按首选科目分类投档录取 |
| **分时分段** | 同一层次按考生总成绩分时分段进行投档 |

## 八、少数民族教育政策

### 8.1 四川少数民族加分政策

四川省对民族地区考生给予高考加分照顾，实施范围为**"三州十七县两区"**：

#### 实施区域

| 范围 | 具体区域 |
|:---|:---|
| **三州** | 甘孜州、阿坝州、凉山州 |
| **十七县** | 峨边县、马边县、米易县、盐边县、石棉县、北川县、平武县、汉源县、宝兴县、兴文县、宣汉县、叙永县、古蔺县、筠连县、珙县、屏山县、荥经县 |
| **两区** | 仁和区、金口河区 |
| **攀枝花东区/西区** | 单独加分标准 |

#### 加分标准（2025年）

| 区域 | 报考部委属/外省属高校 | 报考省属高校 |
|:---|:---|:---|
| **三州十七县两区（少数民族）** | +20分 | 本一批专业+25分，其他+50分 |
| **三州十七县两区（汉族）** | +10分 | 本一批专业+10分，其他+25分 |
| **攀枝花东区/西区（少数民族）** | — | +10分 |
| **攀枝花东区/西区（汉族）** | — | +5分 |

> **来源**: [中国教育在线](https://gaokao.eol.cn/si_chuan/dongtai/202505/t20250519_2669070.shtml)、[成都本地宝](https://m.cd.bendibao.com/edu/190430.shtm)

#### "三统一"要求

从2025年起，享受民族地区加分的考生须符合：
1. **高中阶段在实施区域内具有连续三年完整户籍**
2. **高中阶段在实施区域内具有连续三年完整学籍**
3. **高中阶段在实施区域内实际就读三年**
4. 学籍与实际就读中学应一致
5. 如户籍地与学籍/就读地加分分值不同，**按就低原则**加分

> **关键**：成都市各区**不在**三州十七县两区范围内。因此，在成都就读的少数民族考生（包括布依族），**一般不享受四川少数民族加分**——除非高中回三州十七县两区就读并满足三统一条件。

#### 2026年政策变化趋势

四川省将在2026年后对国家通用语言已普及、教育水平大体相同的县区**逐步缩减或取消**少数民族加分。[[来源: gk100.com]](https://www.gk100.com/read_742559.htm)

### 8.2 贵州少数民族加分政策（布依族适用）

贵州省对少数民族考生实行**分区分类加分**政策，布依族作为贵州世居少数民族之一，适用此政策。

#### 区域分类

| 区域 | 2024-2025年加分 | 2026年起 |
|:---|:---|:---|
| **一类区域** | 已取消（2024年起取消） | — |
| **二类区域** | 符合"三统一"条件，+5分 | 取消该项目加分 |
| **三类区域** | 符合"三统一"条件，+10分 | +5分 |

> **来源**: [学信网](https://gaokao.chsi.com.cn/gkxx/zc/ss/202103/20210330/2049482022.html)、[贵阳本地宝](https://m.gy.bendibao.com/edu/75673.shtm)、[澎湃新闻](https://www.thepaper.cn/newsDetail_forward_20596829)

#### "三统一"条件

考生高中阶段：
1. **三年户籍**在贵州省同一县（市、区、特区）的区域范围内
2. **三年完整学籍**在同一区域
3. **实际就读三年**在同一区域

> **布依族家庭关键提示**：如在成都读高中再回贵州高考，因不满足"三统一"（高中阶段户籍/学籍/就读地不在贵州同一区域），**将无法享受少数民族加分**。若想享受加分，必须在贵州原籍就读高中三年。

#### 政策变化趋势

| 时间 | 一类区域 | 二类区域 | 三类区域 |
|:---|:---|:---|:---|
| 2022-2023 | +5分 | +10分 | +15分 |
| 2024-2025 | **取消** | +5分 | +10分 |
| 2026起 | **取消** | **取消** | +5分 |

> **趋势**：贵州少数民族加分正在**大幅缩减**，到2026年仅三类区域保留5分加分。布依族家庭不应过度依赖加分策略，应以实力为主。

### 8.3 民族预科与民族班

#### 民族预科

| 项目 | 说明 |
|:---|:---|
| **省属高校少数民族预科** | 四川省属高校面向三州十七县两区少数民族考生，具有连续三年以上当地正式户籍可报考 |
| **预科+本科** | 预科1年+本科4-5年，预科合格后升入本科 |
| **录取线** | 通常低于正常录取线一定分数 |

#### 民族班

部分高校设有民族班，专门招收少数民族学生，录取分数线可适当降低。

> **注意**：民族预科和民族班主要面向三州十七县两区等实施区域。在成都就读的布依族考生，如户籍不在实施区域，一般不符合报考条件。

|:---|:---|:---:|
| **A: 成都全程就读→四川高考** | 在成都读小学到高中，参加四川高考。无少数民族加分，但享受成都优质教育 | ★★★★ |
| **B: 成都读小学初中→贵州读高中→贵州高考** | 初中后回贵州原籍读高中三年，满足三统一，享受少数民族加分 | ★★★ |
| **C: 成都全程就读→回贵州高考** | 在成都读高中但回贵州高考。不满足三统一，无加分，且需适应贵州考卷 | ★★ |
| **D: 贵州全程就读→贵州高考** | 全程在贵州就读，享受加分，但教育资源不如成都 | ★★ |

> **核心建议**：策略A（成都全程就读）通常是最优选择——成都教育质量远高于贵州大部分地区，加分5-10分的优势不足以弥补教育质量差距。除非户籍在贵州三类区域且成绩处于临界状态，加分可能起到决定性作用时，才值得考虑策略B。

### 9.1 随迁子女异地高考

#### 四川省随迁子女高考政策

凡父母在四川省有合法稳定职业和住所（含租赁），在父母就业和居住地具有**高中阶段学籍和3年完整学习经历**且符合普通高考其他报名条件的随迁子女，可在就读地报名参加普通高考。

**申请材料**：
- 房产证或父母暂（居）住证明
- 父母合法稳定职业证明
- 在当地就读年限和取得学籍的证明

贵州省对外来人员随迁子女报考普通高等学校有详细规定，符合以下条件之一可在贵州参加高考（仅列出与本文目标读者相关的条件，完整条件详见官方文件）：

### 9.4 布依族家庭路径建议

> **注**：本节为跨省路径视角的分析，少数民族政策视角详见§8.4，决策树视角详见§15.5。

#### 情景分析

假设：孩子目前在成都就读小学，父母一方为布依族，户籍在贵州。

| 阶段 | 推荐路径 | 理由 |
|:---|:---|:---|

| **高中** | **路径分歧点**（见下表） | 需根据成绩和家庭情况决策 |
| **高考** | 按高中就读地决定 | 需提前确认报名资格 |

#### 高中阶段路径决策

| 条件 | 推荐路径 | 理由 |
|:---|:---|:---|
| **成绩优秀（能考成都一流高中）** | 成都读高中→四川高考 | 成都一流高中出口远优于贵州加分5-10分的收益 |

| **成绩一般** | 成都读高中→四川高考 | 成都高中教育质量更有保障，即使无加分，本科率也可能更高 |

> **最终建议**：除非成绩处于关键分数线临界点且贵州加分能起到决定性作用，否则**建议全程在成都就读**。加分政策正在全面缩减，不应作为长期策略依赖。

### 12.4 远期规划框架

#### "三个地基"理论

| 地基 | 说明 | 家庭可控度 |
|:---|:---|:---:|
| **阅读量与知识面** | 广泛阅读是所有学科的基础 | ★★★★★ |
| **学习习惯与自律** | 自律是长期学习的核心竞争力 | ★★★★★ |
| **心理韧性与抗挫折** | 面对失败和压力的恢复能力 | ★★★★★ |

> **核心观点**：孩子的未来不会因为入读某所普通学校就"变坏"，也不会因为入读名校就"自动变好"。**决定性变量不是学校标签，而是家庭教育的投入度、孩子的学习习惯和心理韧性。** 小学阶段最重要的任务不是择校，而是培养这三个地基。

#### 长期规划时间轴

| 时间节点 | 关键决策 | 准备工作 |
|:---|:---|:---|
| **幼儿园大班** | 小学选择 | 了解划片政策、评估民办 |
| **小学三年级** | 学习习惯定型期 | 确保阅读/时间管理/预习复习习惯 |
| **小学五年级** | 小升初准备 | 了解划片/摇号/民办政策 |
| **初一** | 初中三年规划 | 确定目标高中、了解指标到校条件 |
| **初二** | 关键转折期 | 防止成绩下滑、关注心理变化 |
| **初三** | 中考冲刺 | 志愿填报策略、指标到校申请 |
| **高一** | 新高考选科 | 了解专业要求、评估自身优势 |
| **高二** | 学业水平考试 | 完成合格考、准备选择考 |
| **高三** | 高考冲刺+志愿填报 | 强基/专项计划评估、志愿模拟 |

> **初二是最关键的转折点**——很多孩子在初二出现成绩分化，主要原因是难度提升+青春期心理变化。家长应在此阶段加强关注和支持。

## 十三、常见问题（FAQ）

**Q1：孩子在成都读小学，户籍在贵州，可以在成都参加高考吗？**

可以，但需满足四川省随迁子女条件：父母在四川有合法稳定职业和住所，孩子在成都有高中阶段3年完整学籍和学习经历。但注意：在成都参加四川高考，**不享受贵州少数民族加分**。

**Q2：布依族加分值得回贵州读高中吗？**

取决于成绩水平和加分金额。2024-2025年贵州三类区域加分10分，2026年起仅5分。如果孩子成绩在成都能考一流高中（列五/华西等），成都教育质量的优势远大于几分的加分收益。仅当成绩处于关键分数线临界点时，加分才有决定性意义。

**Q3：新高考选科应该选物理还是历史？**

从专业覆盖面看，"物理+化学"组合可报考95%以上专业。除非有明确的文科方向偏好（如法学、文学、历史等），一般建议选物理。选科应综合考虑：①专业兴趣 ②学科成绩 ③学校师资 ④竞争策略。

**Q4：指标到校和中考硬考哪个更好？**

指标到校的优势在于无需中考即可进入优质高中，但需要校内排名前列且中考达普高线。如果孩子成绩在学校名列前茅，指标到校是更稳妥的选择。如果成绩波动大或学校名额少，中考硬考可能更灵活。

**Q5：在成都读初中，回贵州读高中，学籍怎么转？**

需在初二第一学期前完成转学（影响指标到校资格），并办理学籍跨省转移手续。具体流程：成都初中开具转学证明→贵州高中接收→两地教育局学籍系统操作。建议提前一学期开始办理。

**Q6：四川新高考和贵州新高考有什么区别？**

两省均采用3+1+2模式，满分750分，基本框架一致。主要区别在于：①实施时间（贵州2024年首考，四川2025年首考）②具体赋分规则可能略有差异 ③录取分数线和竞争程度不同 ④少数民族加分政策不同。

**Q7：孩子被统筹到非理想学校怎么办？**

"统筹"不等于"最差"，是学位紧张时的协调安排。建议：①实地考察学校，了解校风和师资 ②积极融入，配合学校 ③将精力投入家庭教育——学校标签不决定终生，学习习惯和心理韧性才是关键变量。

**Q8：随迁子女在成都上学需要什么手续？**

每年4月1日-30日在"成都市义务教育招生入学服务平台"申请随迁子女就学。需提供：父母居住证明、稳定职业证明、学籍证明等。审核通过后由教育局统筹安排学位。

## 十五、战略决策框架：朴素家长决策指南

> **本章是整份指南的决策核心。** 假设您是一位对教育体系一无所知的家长，本章将带您一步步做出战略决策。

### 15.1 朴素家长必知的10条底层逻辑

| 序号 | 底层逻辑 | 解释 |
|:---|:---|:---|
| **1** | **学校标签≠孩子未来** | 决定孩子未来的不是学校名气，而是学习习惯、心理韧性和家庭教育投入度 |
| **2** | **分数是硬通货** | 中考、高考都是分数优先。无论走哪条路径，最终都需要分数支撑 |
| **3** | **多路径并行降低风险** | 不要只走一条路。同时准备2-3条路径（如划片保底+指标到校+中考统招） |
| **4** | **提前规划优于临时冲刺** | 教育规划应从幼儿园开始，至少提前3年了解下一阶段政策 |
| **5** | **经济投入要量力而行** | 全公办路径7万即可完成15年教育，民办可达150万。高投入≠高产出 |
| **6** | **初二是关键分水岭** | 成绩分化主要发生在初二，难度提升+青春期双重挑战 |
| **7** | **选科决定专业方向** | 高一选科直接影响大学专业选择。"物理+化学"覆盖95%专业 |
| **8** | **加分政策正在全面缩减** | 少数民族加分、艺体特长等特殊政策逐年收紧，不应长期依赖 |
| **9** | **指标到校是性价比之王** | 免中考或只需达普高线，零额外成本，选择小规模初中可提高获取概率 |
| **10** | **家庭教育是最大变量** | 阅读习惯、时间管理、心理韧性——这三个"地基"比任何学校都重要 |

### 15.2 按年龄段决策树

#### 幼儿园阶段（3-6岁）决策

```
孩子在成都 →
  ├─ 户籍在成都 → 报公办园（摇号）+ 普惠民办（备选）
  │   └─ 未中签 → 报其他民办园（自费）
  ├─ 户籍不在成都 → 报普惠民办 + 其他民办
  └─ 核心任务：培养安全感、社交能力、好奇心
      └─ 不需要学知识！不需要提前识字算术！
```

#### 小学阶段（6-12岁）决策

```
孩子要上小学 →
  ├─ 户籍在成都 → 公办划片入学（确定性最高）
  │   ├─ 想尝试民办 → 同时报名民办摇号（不中签不影响公办）
  │   └─ 在民办一贯制学校就读 → 可考虑直升
  ├─ 户籍不在成都 → 4月申请随迁子女就学
  └─ 核心任务（比择校更重要！）：
      ├─ 一年级：建立作业习惯、阅读习惯
      ├─ 二年级：巩固时间管理能力
      ├─ 三年级：开始独立思考、错题整理
      ├─ 四年级：关注学习兴趣变化、发现特长
      ├─ 五年级：了解小升初政策、确定策略
      └─ 六年级：执行小升初方案
```

#### 小升初决策树

```
孩子要上初中 →
  ├─ 在民办一贯制学校就读且满意？
  │   ├─ 是 → 申请直升（确定性高，但放弃公办机会）
  │   └─ 否 → 继续以下路径
  ├─ 有明确艺体特长？（注意：此渠道正在收紧）
  │   ├─ 是 → 艺体特长+划片保底
  │   └─ 否 → 继续以下路径
  ├─ 核心策略：划片保底 + 可选博弈
  │   ├─ 保守型：只走划片（最省心）
  │   ├─ 稳健型：划片 + 民办摇号
  │   ├─ 进取型：划片 + 大摇号 + 民办摇号
  │   └─ 关键：大摇号中签后不再参加其他录取！
  └─ 选校考量：
      ├─ 小规模初中 → 指标到校竞争压力小（推荐！详见§5.2）
      ├─ 大规模名校初中 → 师资好但指标竞争激烈
      └─ 完全中学（有初中+高中）→ 可校内直升
```

#### 中考决策树

```
孩子面临中考 →
  ├─ 成绩在年级前列？
  │   ├─ 是 → 申请指标到校（市级/区级/校内，详见§5.2）
  │   │   ├─ 获得 → 无需中考（区级/校内）或只需达普高线（市级）
  │   │   └─ 未获得 → 继续中考
  │   └─ 否 → 继续中考
  ├─ 有艺体特长？
  │   ├─ 是 → 报名第一批次艺体特长生（6月16-17日，详见§6.4）
  │   └─ 否 → 不报
  ├─ 有特殊志向（科技/航空/球类）？
  │   ├─ 是 → 关注项目班招生（第一批次第7类，详见§6.4）
  │   └─ 否 → 不报
  ├─ 中考出分后 →
  │   ├─ 分数≥一流高中线 → 填报四七九/列五/华西等
  │   ├─ 分数≥二流高中线 → 填报49中/英才等
  │   ├─ 分数≥普高线 → 填报三流高中或民办高中
  │   ├─ 分数≥职普融通线 → 填报职普融通班
  │   └─ 分数<职普融通线 → 中职+对口高考路径
  └─ 户籍不在5+2？
      └─ 注意：只能走调招（名额少）或回户籍地中考
```

#### 高考路径决策树

```
孩子面临高考 →
  ├─ 成绩在年级前5%？
  │   ├─ 对基础学科有兴趣 → 报强基计划（4-5月报名，详见§7.3-§7.4）
  │   ├─ 综合素质好 → 报综合评价（4-5月报名，详见§7.5）
  │   └─ 都不报 → 走普通高考
  ├─ 有五大学科竞赛国家集训队资格？
  │   └─ 是 → 申请竞赛类保送（极少数人，详见§7.6）
  ├─ 在成都外国语学校且成绩优异？
  │   └─ 是 → 申请外语类保送（详见§7.6）
  ├─ 有艺术特长？
  │   ├─ 是 → 参加省级统考（12月-1月）+ 高考文化课（详见§7.8）
  │   └─ 否 → 不走此路
  ├─ 有体育特长（二级运动员以上）？
  │   ├─ 是 → 考虑高水平运动队或体育类专业（详见§7.8）
  │   └─ 否 → 不走此路
  ├─ 有志从军/从警且身体条件好？
  │   ├─ 是 → 报军事/公安类（提前批A段，详见§7.9）
  │   └─ 否 → 不走此路
  ├─ 家庭经济条件好+英语好？
  │   ├─ 想出境 → 港澳高校（独立报名，详见§7.11）
  │   ├─ 想国际化但不出境 → 中外合作办学（详见§7.11）
  │   └─ 不需要 → 走普通高考
  ├─ 家庭经济困难+有志从教/从医？
  │   ├─ 愿回基层服务6年 → 公费师范/定向医学生（详见§7.10）
  │   └─ 不愿 → 走普通高考
  ├─ 户籍在农村贫困地区？
  │   ├─ 是 → 关注国家/地方/高校专项计划（详见§7.3）
  │   └─ 否 → 不适用
  ├─ 少数民族+户籍在实施区域？
  │   ├─ 满足三统一 → 享受少数民族加分（详见§八）
  │   └─ 不满足 → 无加分，走普通高考
  └─ 默认路径 → 普通高考（本科批B段，45个平行志愿）
```

| **4-5月** | 强基计划/综合评价报名 | 错过特殊招生机会 | 高三 |
| **6月下旬-7月上旬** | 强基计划校考 | 错过强基录取机会 | 高三 |
| **6月中下旬** | 综合评价校测 | 错过综合评价录取机会 | 高三 |
| **6月7-9日** | 高考 | 无法上大学 | 高三 |
| **高一第一学期** | 新高考选科 | 选科后难以更改 | 高一 |

### 15.4 关键约束清单（硬门槛，不可变通）

| 约束 | 影响路径 | 说明 |
|:---|:---|:---|
| **户籍** | 小学入学、中考统招/调招、专项计划、少数民族加分 | 户籍决定了很多路径的资格，提前迁户需谨慎 |
| **学籍** | 指标到校、随迁子女高考 | 指标到校需3年完整学籍，中途转学可能丧失资格 |
| **年龄** | 少年班 | 中科大少年班≤2009年生、创新试点班≤2008年生；西交大初三≤2009年9月1日生、高二≤2008年生 |
| **身体条件** | 军事/公安/飞行技术/航空实验班 | 视力、身高、体重等硬性标准，不可变通 |
| **政审** | 军事/公安类 | 家庭成员政治背景审查 |
| **经济条件** | 民办学校、艺术类、港澳高校、中外合作办学 | 高费用路径需家庭经济支撑 |
| **三统一** | 少数民族加分 | 高中三年户籍+学籍+实际就读地在同一区域 |
| **服务期** | 公费师范/定向医学生 | 毕业后必须回基层服务6年 |
| **选科** | 高考专业选择 | "物理+化学"覆盖95%专业，选历史则限文科专业 |

### 15.5 布依族家庭专属决策框架

> **注**：本节为决策树视角，政策详解详见§8.4，跨省路径详见§9.4。

```
布依族家庭（户籍贵州，孩子在成都就读）→

  问：孩子成绩在成都能考一流高中吗？
  ├─ 是（成绩优秀）→
  │   └─ 策略A：成都全程就读→四川高考
  │       └─ 理由：成都一流高中出口 >> 贵州5-10分加分收益
  │
  ├─ 否（成绩中等）→
  │   └─ 问：贵州户籍在三类区域吗？
  │       ├─ 是 → 问：成绩在一本线临界点吗？
  │       │   ├─ 是 → 策略B：回贵州读高中→贵州高考（加分可能起决定作用）
  │       │   └─ 否 → 策略A：成都全程就读（加分收益不足以弥补教育质量差距）
  │       └─ 否（一/二类区域）→
  │           └─ 策略A：成都全程就读（一/二类区域加分2026年起取消或仅5分）
  │
  └─ 问：孩子有艺体特长吗？
      ├─ 有 → 可考虑艺术/体育类高考路径（不受加分政策影响）
      └─ 无 → 走普通高考路径

  ⚠ 重要提醒：
  1. 贵州少数民族加分2026年起仅三类区域保留5分
  2. 回贵州读高中需满足"三统一"才能享受加分
  3. 在成都读高中→回贵州高考 = 不满足三统一 = 无加分
  4. 加分政策正在全面缩减，不应作为长期策略依赖
```

### 15.6 家长自检清单

> **请家长对照以下清单，逐项确认自己是否已做好充分准备。**

#### 幼儿园阶段自检

- [ ] 了解所在区公办幼儿园名单和摇号政策
- [ ] 知道幼儿园报名时间和所需材料
- [ ] 理解幼儿园阶段重点是习惯培养而非知识学习
- [ ] 已开始每日亲子阅读

#### 小学阶段自检

- [ ] 了解户籍对应的小学划片范围
- [ ] 知道民办小学报名时间和流程
- [ ] 如户籍不在成都，已了解随迁子女申请流程
- [ ] 已建立孩子每日作业+阅读的固定时间表
- [ ] 知道初二分化的风险，已开始培养自主学习习惯

#### 小升初自检

- [ ] 了解孩子所在小学的对口初中（单校划片）或片区初中（多校划片）
- [ ] 知道大摇号、民办摇号、直升的报名时间和互斥规则
- [ ] 了解指标到校制度及所在初中的指标名额
- [ ] 已评估是否需要选择小规模初中以提高指标到校概率
- [ ] 知道艺体特长渠道正在收紧

#### 中考自检

- [ ] 了解统招生vs调招生的区别及对孩子的影响
- [ ] 知道第一批次7类录取顺序及孩子可走的特殊渠道
- [ ] 了解指标到校申请条件，孩子是否具备资格
- [ ] 知道2025年中考改革：取消省重线、合并批次、平行志愿
- [ ] 已了解职普融通作为安全网选项
- [ ] 知道中考志愿填报时间和流程

#### 高考自检

- [ ] 了解新高考3+1+2模式及赋分制
- [ ] 知道12种选科组合及专业覆盖面
- [ ] 已帮孩子确定选科方向（高一第一学期必须决定）
- [ ] 了解强基计划、综合评价等特殊招生渠道
- [ ] 知道各特殊渠道的报名时间和条件
- [ ] 了解志愿填报的院校专业组模式
- [ ] 如有少数民族背景，已了解加分政策和三统一要求
- [ ] 如考虑跨省高考，已了解两地随迁子女政策
- [ ] 已了解全部18条高考升学路径（见§7.12）
