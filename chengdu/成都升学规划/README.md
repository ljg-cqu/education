## 项目说明

本知识体系围绕成都（以成华区为重点）的升学路径展开，覆盖从幼儿园到大学的全链路政策、择校策略与行动指南。内容基于公开政策文件、官方招生平台及家长口碑信息整理，**仅供参考**。

教育政策动态变化，请以以下官方渠道发布的最新信息为准：
- 成都市义务教育招生入学服务平台（yjrx.cdeduypt.cn）
- 四川省教育考试院（sceea.cn）
- 成都市教育局、各区教育局官方微信公众号
